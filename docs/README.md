# Learning Repository — Docs

This directory contains supporting documentation for the `omatvis/learning` repo.

## Structure

| Path | Purpose |
|------|---------|
| `roadmap.md` | Overall curriculum, modules, small projects, capstone checkpoints |
| `conventions/` | Coding and architecture conventions per technology/style |
| `templates/` | Markdown templates for lessons, exercises, and projects |
| `references/` | Curated resources and glossary |

## How to use this repo

1. Read `roadmap.md` to understand the overall learning sequence.
2. Work through `learning-path/` modules in order.
3. For each module, open a Copilot Chat in this repo and ask: _"Teach me [topic] and give me exercises that fit this repo structure."_
4. Implement on a branch: `exercise/<module>-<n>`.
5. Open a PR into `main` and add Copilot as a reviewer.
6. Apply feedback, iterate, and merge.
