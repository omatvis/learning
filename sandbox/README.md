# Sandbox

> Throwaway experiments — code here is **not** reviewed unless you explicitly request it.

Use this space for quick spikes, trying out libraries, or testing ideas before incorporating them into a real project.

## Sub-directories

| Directory | Purpose |
|-----------|---------|
| `dotnet/` | .NET / C# scratch space |
| `frontend/` | React / TypeScript / JavaScript scratch space |

## Rules

- Code here does not need to meet the conventions in `docs/conventions/`.
- Do not add sandbox code to PRs for coaching review (unless you want feedback on it).
- Clean up periodically — delete experiments that are no longer useful.
