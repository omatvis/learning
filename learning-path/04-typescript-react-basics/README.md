# 04 — TypeScript + React Basics

> **Goal:** Build typed React components that fetch data from an API and handle UI state correctly.

## Topics covered

- TypeScript strict mode: types, interfaces, generics, utility types
- React functional components and hooks (`useState`, `useEffect`, `useContext`)
- Custom hooks
- Server state with React Query
- Forms and validation
- Accessibility basics

## How to use this module

> "Teach me React hooks and give me exercises."

## Exercises

See the `exercises/` folder.  
Submit each exercise as a PR from branch `exercise/04-react-<n>`.
