# 02 — Web API Basics

> **Goal:** Build and test ASP.NET Core HTTP APIs using both Minimal APIs and Controllers.

## Topics covered

- Minimal API vs MVC Controllers — when to use each
- Routing, HTTP methods, status codes
- Request/response models (DTOs, records)
- Input validation and error handling (`ProblemDetails`)
- Middleware and filters
- OpenAPI / Swagger
- Integration testing with `WebApplicationFactory`

## How to use this module

> "Teach me Minimal APIs in .NET and give me exercises."

## Exercises

See the `exercises/` folder.  
Submit each exercise as a PR from branch `exercise/02-webapi-<n>`.
