# m5 — React Frontend

> Build a React/TypeScript frontend that consumes the capstone API.

## Goals

- Set up the Vite + React + TypeScript project in `frontend/`.
- Implement all user-facing features.
- Handle auth, loading, and error states correctly.

## Tasks

- [ ] Scaffold Vite project in `projects/capstone/capstone-app/frontend/`.
- [ ] Set up React Query for server state.
- [ ] Implement authentication flow (login, register, token refresh).
- [ ] Implement main feature pages.
- [ ] Handle loading and error states in every data-fetching component.
- [ ] Add basic accessibility (semantic HTML, ARIA labels).
- [ ] Wire up with `docker-compose.yml` for a one-command local start.

## Acceptance criteria

- [ ] TypeScript strict mode enabled; no `any`.
- [ ] All pages handle loading and error states.
- [ ] `docker-compose up` starts both backend and frontend.
- [ ] Follows `docs/conventions/react.md` and `docs/conventions/typescript.md`.
