# 01 — C# / .NET Basics

> **Goal:** Be comfortable with the C# language features you'll use in every subsequent module.

## Topics covered

- Types, nullability, and nullable reference types
- Collections and LINQ
- Async/await and `Task<T>`
- Records, pattern matching, and modern C# syntax
- Dependency Injection basics

## How to use this module

Ask Copilot Chat (in this repo) to teach each topic and provide exercises:

> "Teach me LINQ in C# and give me exercises that fit this repo structure."

## Exercises

See the `exercises/` folder for exercise specs.  
Submit each exercise as a PR from branch `exercise/01-csharp-<n>`.
