# 05 — Small Projects

> **Project-based learning**: apply what you learned in modules 01–04 by building small, focused projects.

## Rules

- Each small project lives in `projects/small/sp-XX-<name>/`.
- Use the appropriate template from `projects/templates/` as a starting point.
- Open one PR per meaningful change; do not batch everything into a single PR.
- The PR description must follow `.github/pull_request_template.md`.
