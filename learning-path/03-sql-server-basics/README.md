# 03 — SQL Server Basics

> **Goal:** Design schemas, write queries, and understand transactions and indexing in SQL Server.

## Topics covered

- Schema design and table relationships
- SELECT, INSERT, UPDATE, DELETE
- JOINs and subqueries
- Indexes and query plans (basics)
- Transactions and isolation levels
- Stored procedures (when to use and when not to)
